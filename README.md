# placeholderfiles
A quick script to create 0 byte copies of files as a placeholder.

# Install
* Download repo
* Run `npm install`
* Run `npm link`

# Usage
* Switch to source directory
* Run `placeholderfiles -d /path/to/destination` (no trailing slash)
